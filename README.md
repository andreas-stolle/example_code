# Selected Software Engineering Projects

A curated selection of my programming projects. Only the **core source files** are included to keep the repository concise and easy to review.

## Structure

- `maze_gen_solve/` — Maze generator and solver in C++ (runnable)
- - `os_components/` — Operating system components in C (shell, memory allocator, filesystem)
- `ijvm_interpreter/` — IJVM bytecode interpreter in C (runnable)
- `secure_chat_server/` — Secure client–server chat application in C
- `asm_diff/` — File differencing tool implemented in x86 assembly (runnable)
